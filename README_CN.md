# 词悦

[English](./README.md) | 中文

支持 Android 的 mdict 词典。

## 特点

* 📄 支持 MDX/MDD
* 🎨 支持 Material You
* 🔈 支持朗读
* 🔖 收藏单词到单词本
* 📙 添加多个词典并在它们之间切换

## 权限

### INTERNET（互联网）

词悦使用 WebView 显示单词，WebView 即使显示本地数据也需要 INTERNET 权限才能正常工作。

### READ_EXTERNAL_STORAGE / MANAGE_EXTERNAL_STORAGE（管理外部存储空间）

词悦需要该权限去读取 mdd 文件（mdx 文件的资源）。(如图片、JavaScript、CSS 等）。
